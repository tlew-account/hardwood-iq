Import multiple questions at once via JSON.
JSON Format

Paste or upload a JSON array. Required per question: topic_name, question_text, options, correct_option. Optional: question_type ("text"/"video"/"image"), video_url, image_url, image_alt, points.

  {
    "topic_name": "NBA History",

    "points": 10

    "topic_name": "Famous Plays",
